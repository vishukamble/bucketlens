    before you browse
    how images load
    Images stream through localhost on demand. Nothing is downloaded or saved to disk. Flask proxies S3 bytes directly to your browser.
    S3 → Flask (localhost) → your browser
          ↑
     nothing saved
    S3 costs
    Every image you view is one S3 GET request. Browsing 500 thumbnails = 500 GET requests. S3 GET pricing: ~$0.0004 per 1,000 requests.
    Use the size slider to load fewer thumbnails at once. Folders help too — navigate rather than scroll through thousands of files.
    estimated cost for this session
    viewing 100 thumbnails  →  ~$0.00004
viewing 1,000 files     →  ~$0.0004
downloading 1GB         →  ~$0.09
    // S3 pricing varies by region and account tier
    // these are us-east-1 standard rates

    how it works
    Images are never saved to disk. Flask proxies S3 bytes in 8KB chunks to your browser. RAM is freed after each response. Thumbnails are cached on disk for repeat visits.
    S3 → Flask → resize → cache → browser
     ↑                    ↑
8KB chunks          400×400 JPEG
                    on disk
